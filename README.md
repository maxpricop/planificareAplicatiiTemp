# planificareAplicatiiTemp

Depozit de exercițiu folosit la laborator pentru a demonstra utilizarea de bază a Git și GitHub.

## Prototipuri Figma

- UI Libraries: https://www.figma.com/design/WqKLaLxEzx1SHB0fjsmebw/Laborator-UI-Libraries

## Scop

- Acest repository este un spațiu simplu de laborator care arată că studentul știe să inițializeze un repo, să facă commit și să împingă modificările către GitHub.

## Conținut

- `LICENSE` — licența proiectului
- `README.md` — acest fișier
- `folder1/lab1.txt` — fișier de exemplu pentru laborator

## Instrucțiuni rapide

1. Clonează repository-ul:

```bash
git clone https://github.com/maxpricop/planificareAplicatiiTemp.git
cd planificareAplicatiiTemp
```

2. Fă o modificare (de ex. editează `folder1/lab1.txt` sau `README.md`), apoi adaugă, comite și împinge:

```bash
git add .
git commit -m "Actualizare README pentru exercițiu de laborator"
git push origin main
```
